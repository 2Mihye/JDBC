SelectOne
	selectOne : 주로 단일 행(레코드) 검색하는 데 사용
		일반적으로 WHERE절을 사용하여 조건을 지정하고, 결과로 단일 객체나 레코드를 반환한다. 하여 결과가 없거나 여러 결과가 나오는 경우 첫 번째 결과만 주로 반환한다.
		주요키(Primary Key) 값으로 검색하거나 고유한 결과를 필요로 하는 경우에 사용된다.
		
	selectState.setString(값의 위치, 임의값);
	if(result.next()) : 다음 행으로 이동하고, 다음 행이 존재하는 경우에만 if 블록 내의 코드를 실행한다.
